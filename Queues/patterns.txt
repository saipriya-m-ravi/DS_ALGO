“Every perfect number can be represented as X concatenated with its reverse, where X consists only of 1s and 2s. 
I generate X in BFS order using a queue and mirror it to get the Ath perfect number.”

-> Monotonic deque

Sliding Window Maximum (queue/deque)
We maintain a deque of indices, such that. Elements are in decreasing order of values
Front of deque = max of current window

🧱 Rules of the Deque
For each index i:

1️⃣ Remove out-of-window elements
2️⃣ Maintain decreasing order
3️⃣ Add current index
4️⃣ Record answer

“I use a monotonic decreasing deque to maintain candidates for maximum in each sliding window, achieving O(N) time.”
