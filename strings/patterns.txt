🔴 10. Backtracking on Strings
🔎 Recognition clues

palindrome partitioning

🔥 Example problems

Palindrome partitioning

-> If problem says:

✔ cyclic shifts
✔ rotations
✔ matching strings

Immediately think: string + string trick

-> A[i] = A[i % k]   for all i means the string is built by repeating the first k characters.

->  “How do you find smallest repeating block?”
Build LPS array and compute N - LPS[N-1].

Whenever you see:

✅ repeating substring
✅ smallest pattern
✅ string constructed by repetition
✅ periodic string

Immediately think: ⭐ KMP LPS

Period = length − longest prefix which is also suffix

-> Make string palindrome
We want the longest prefix of A that is already a palindrome
answer = N - (length of longest palindromic prefix)

How to find longest palindromic prefix FAST? 
1. Build string S = A + "#" + reverse(A)
2. Then compute:
👉 LPS array (KMP prefix function)
The last value of LPS =
⭐ longest prefix of A matching reversed suffix
⭐ i.e. longest palindromic prefix

-> Whenever you see:

✅ insert only at front
✅ insert only at back
✅ minimum chars to make palindrome

Immediately think:

⭐ KMP with A + '#' + reverse(A)

-> Z[i] = length of longest substring starting at i
which matches prefix of string.

	          LPS	               Z
Meaning	prefix==suffix	prefix match from position
Used in	KMP search	pattern finding / substring tricks
Direction	looks backward	looks forward
Intuition difficulty	harder initially	easier conceptually

✅ Use LPS when problem says:

✅ Use Z-algorithm when problem says:
